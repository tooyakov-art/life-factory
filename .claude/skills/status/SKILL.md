# Статус проекта

## Инструкции

1. Прочитай `package.json` — покажи версии зависимостей
2. Запусти `npm run build` — покажи результат
3. Посчитай файлы:
   - Компоненты: `src/components/**/*.tsx`
   - Страницы: `src/app/**/page.tsx`
   - Утилиты: `src/utils/*.ts`
   - Типы: `src/types/*.ts`
   - Stores: `src/store/*.ts`
4. Покажи структуру в таблице
5. Проверь наличие TODO/FIXME/HACK в коде через grep
6. Покажи итог: что готово, что нужно доделать
